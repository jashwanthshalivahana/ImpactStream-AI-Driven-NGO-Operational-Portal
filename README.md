# NayePankh Foundation - AI-Integrated Web Prototype

A modern, high-performance web platform designed to streamline operations for the NayePankh Foundation. This project was developed as a prototype to modernize volunteer management, automate donor engagement, and provide an interactive AI-driven assistant for stakeholders.

## Key Features

* **Interactive 3D UI:** A cinematic, glassmorphism-inspired interface that elevates the organization's digital presence.
* **AI Virtual Assistant:** Integrated with the Groq API, this assistant is context-aware, providing instant answers about NGO operations (12A/80G certifications, impact data) and handling administrative escalation.
* **Automated Data Management:** Volunteer applications are captured and automatically sorted into local data structures, eliminating manual entry.
* **Student Partnership Program:** A gamified referral system with real-time tracking, allowing students to earn rewards and track their impact.
* **Impact Dashboard:** A visual representation of the NGO's milestones and organizational statistics.
